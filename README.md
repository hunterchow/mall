# mall

> A vue2.0 fulltack project

## 使用技术包括： ##
1. axios和后端交互
2. vuex做状态管理
3. vue-router项目路由
4. Node.js/express提供后台接口
5. vue-lazyload做图片懒加载
6. MongoDB做数据库
7. 前后端分离
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
